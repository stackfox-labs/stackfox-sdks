# @stackfox/sdk-luau

The Luau SDK for StackFox.

Use this package in your Roblox game to send events and read or write records through the StackFox platform.

For more information on how to use the SDK, please refer to the [documentation](https://stackfox.dev/docs/sdk/luau/overview).
